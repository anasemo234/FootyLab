# Footy Lab

This app is a football jersey online store. Come shop and pick out what you like!!
* AAU - Should be able to add a product to sell
      - Able to edit your product
      - write reviews on products
      

## Technologies Used

* Express.js
* Node.js
* Bootstrap
* MongoDB
* HTML
* CSS
* JavaScript
* Google Fonts
* Mongoose
